## Install Language

Install Elm language via [installer](https://guide.elm-lang.org/install.html)

Verify in terminal:

```
elm -v   # should be 0.18.x
```

## Install Test Framework

(requires node)

```
npm install elm-test -g
```

## Download Exercises

```
git clone https://github.com/jaketrent/elm-kata-map
```

## Run Tests

```
cd elm-kata-map && \
elm-test
```
